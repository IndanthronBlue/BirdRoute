图片来源：Wikimedia/Wikipedia；鸟声来源：xeno-canto。系统会优先用学名和英文名检索，中文名作为兜底；大多数情况下不需要 token。

获取 eBird API Token

登录 eBird 后访问：

https://ebird.org/api/keygen

复制生成的 API Key，粘贴到左侧输入框即可使用 eBird 热点图层。

API Key 与你的 eBird 账号绑定，请勿公开分享。

如何获取 xeno-canto API Key

xeno-canto API v3 推荐/要求使用个人 API key。旧的 API v2 在某些情况下仍能访问，但浏览器本地 HTML 更容易遇到跨域、重定向或格式变化问题。

获取步骤

  1. 打开 xeno-canto：https://xeno-canto.org
  2. 注册账号并验证邮箱，或登录已有账号。
  3. 进入个人 Account / Profile 页面。
  4. 查找 API key / API v3 key。第三方库文档也说明 API v3 key 可在 Account Page 获得。
  5. 复制 key，粘贴到本页面左侧“xeno-canto 鸟声 API”输入框。
  6. 再点击鸟点卡片里的“鸟种预习”。

本页面调用方式

GET https://xeno-canto.org/api/3/recordings?query=Butastur%20indicus&key=你的APIkey

优先检索顺序：
1. 学名，例如 Butastur indicus
2. 英文名，例如 Gray-faced Buzzard
3. 中文名兜底

常见问题

  • 网页能搜到，但本页面显示没有鸟声：通常是 API key、CORS、接口版本或查询语法问题，不代表 XC 没有记录。
  • 401 / 403：key 无效、未登录验证、key 复制错误。
  • 无法播放但有链接：点击“打开 xeno-canto 记录”即可在官网播放。
  • 本地 HTML 限制：若浏览器安全策略严格，可以用本地服务器打开，例如 python3 -m http.server。
