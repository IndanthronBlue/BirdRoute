如何获取 xeno-canto API Key

xeno-canto API v3 推荐/要求使用个人 API key。旧的 API v2 在某些情况下仍能访问，但浏览器本地 HTML 更容易遇到跨域、重定向或格式变化问题。

获取步骤

  1. 打开 xeno-canto：https://xeno-canto.org
  2. 注册账号并验证邮箱，或登录已有账号。
  3. 进入个人 Account / Profile 页面。
  4. 查找 API key / API v3 key。第三方库文档也说明 API v3 key 可在 Account Page 获得。
  5. 复制 key，粘贴到本页面左侧“xeno-canto 鸟声 API”输入框。
  6. 再点击鸟点卡片里的“鸟种预习”。

本页面调用方式

GET https://xeno-canto.org/api/3/recordings?query=Butastur%20indicus&key=你的APIkey

优先检索顺序：
1. 学名，例如 Butastur indicus
2. 英文名，例如 Gray-faced Buzzard
3. 中文名兜底

常见问题

  • 网页能搜到，但本页面显示没有鸟声：通常是 API key、CORS、接口版本或查询语法问题，不代表 XC 没有记录。
  • 401 / 403：key 无效、未登录验证、key 复制错误。
  • 无法播放但有链接：点击“打开 xeno-canto 记录”即可在官网播放。
  • 本地 HTML 限制：若浏览器安全策略严格，可以用本地服务器打开，例如 python3 -m http.server。

小红书第三方数据 API

小红书官方开放平台不一定提供通用的全站笔记搜索能力。若需要更稳定地搜索和解析公开笔记，可以使用用户自备的第三方数据 API。

可参考： https://rnote.dev/

当前已按第三方 V2 流程预填默认端点：先搜索笔记，再按 note_id 抓取图文或视频详情。
https://rnote.dev/api/v2/crawler/search/notes
https://rnote.dev/api/v2/crawler/note/image
https://rnote.dev/api/v2/crawler/note/video 
GET /api/v2/crawler/search/notes
Query:
  keyword = 攻略搜索关键词
  page = 1
  sort_type = general
  note_type = 不限 / 普通笔记 / 视频笔记 / 直播笔记
  time_filter = 不限 / 一天内 / 一周内 / 半年内

Header:
  X-API-Key: 你的第三方 API Key

GET /api/v2/crawler/note/image?note_id=...
GET /api/v2/crawler/note/video?note_id=...
在该服务获取 API Key 后，把 Key 填入“内容源接入设置”。BirdRoute 会把鸟点关键词作为 keyword 传入；用户点击“抓取正文”时，再根据搜索结果中的 note_id 调详情接口。请自行确认服务条款、数据授权和使用限额。
